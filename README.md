# Dotfiles

Currently supports:
- Visual Studio Code
- Windows Terminal
- Powershell (with Oh-My-Posh)

## Requirements

You will need to install the above mentioned software

## Usage

After installing the required software, open an elevated PowerShell console and run the script. It will create symlinks from the files in the local repository to the locations where these programs take their configs from.

> [!WARNING]
> You will need to run the `setup.ps1` script in an elevated (administrator) PowerShell terminal
